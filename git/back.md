# 有关回退的操作

在 `git` 中使用 `HEAD` 表示当前版本， 上一个版本就是 `HEAD^`, 上上一个版本就是 `HEAD^^`, 上 100 哥版本可以写成 `HEAD~100`。

> 使用 `git reflog` 查看命令历史， 以便确定要回到哪个版本。
