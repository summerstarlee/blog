# 本地更新 git 仓库的用户密码

当远端服务器更改了用户的密码后，用户在本地 pull / push 时会提示无权限： fatal: Authentication failed for xxxxx.git。 这时用户需要更新已经保存在本地的密码。

windows 系统： 步骤：

  1. 进入： 控制面板>所有控制面板项>凭据管理器。找到该仓库地址，点击右侧下拉图表。
  2. 点击编辑，输入新密码并保存。
